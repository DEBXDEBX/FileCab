To add a file cabinet or folder, type the name of it in the name text input and click the corresponding button.
To shift a file cabinet left, hold down shift and click the file cabinet name.
To delete a file cabinet, folder or note hold down control key and click it
To add an image to a note, place the image in the image folder of this project, type the image Name in the Name text field and press the add image to note button.
