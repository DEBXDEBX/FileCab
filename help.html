To add a Note to your File Cabinet, You must first load or create a File Cabinet. Then You must add a Main Folder and a Sub Folder. Now You can add a Note

To create a File Cabinet

Click on File and then select Create File Cabinet, Enter a name and click add. Then You will be promted to save the path to your New File Cabinet Enter a name for the path and click save.

To load a File Cabinet

Click on File and then select Load File Cabinet. Navigate to the file where you have your File Cabinet. Click on it and click open.

To add a Main Folder

Click on the File Cabinet name, and a Main Folder section will pop up. Click on the small icon beside the word Main Folder. This will bring up a form. Enter a name for your new Main Folder and click add.

To add a Sub Folder

Click on a main folder tab, and a Sub Folder section will pop up. Click the small icon beside the word Sub Folder. This will bring up a form. Enter a Name for your new Sub Folder and click add.

To add a Note

Click a Sub Folder tab, and a note section will pop up. Click the small icon beside the word Note. This will bring up a form for your note. Enter your Note and click add.

To add a Image to a Note.

To add an image to a Note, hold down alt and click on a note, the file picker will promt you for a path. Find your picture and select it and click open. To see the picture that belong's to the Note, Just click on the Note and the picture will pop up. To dismiss the picture just click the picture and it will go away. To remove an image path that no longer exists hold down shift and click on the Note

To enter Delete Mode

Click on Mode and select Read Write Delete

To exit Delete Mode

Click on Mode and select Read and Write

Delete a Main Folder, Sub Folder, or Note

First enter delete Mode. See above. To delete Main Folder or Sub Folder or Note hold down control key and click it. Deleting a Main Folder or Sub Folder will delete everything under it.

To change the File Cabinet name.

Hold down shift and click the File Cabinet Name. Fill in the form and click Rename File Cabinet
